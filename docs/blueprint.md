# **App Name**: Logic Arena

## Core Features:

- Real-Time Duels: Engage in 1v1 real-time aptitude and logical reasoning challenges against other users. Implements a 60-second timer.
- Asynchronous Duels: Take turns with opponents in asynchronous aptitude and logical reasoning challenges, allowing for flexible play.
- Today's Challenges: Access daily sets of logic puzzles and aptitude sprints to enhance skills.
- Category Selection: Choose duel categories such as Aptitude, Logical Reasoning, Data Interpretation, and Classical to focus on specific areas.
- Elo Rating System: Calculate player skill and display Elo ratings for competitive matchmaking. Players gain 10XP for a win, lose 1XP for a loss.
- Performance Analytics: Display detailed post-match analytics, including score, ELO rating change, speed comparison graph, and a stat table comparing average, fastest, and slowest answer times using a tool to determine the insights most important for users.
- Gamification and Progression: Track daily streaks for consecutive logins, reward XP for wins, and display total XP earned.

## Style Guidelines:

- Light Theme (Clarity): Primary color: Ink Black (#1A1A1A) for text and main elements.
- Light Theme (Clarity): Background color: Pale Gray (#F7F7F7) for a subtle base.
- Light Theme (Clarity): Accent color: Success Green (#7CAA8C) for correct answers.
- Dark Theme (Focus): Primary color: Off-White (#F0F0F0) for text on a dark background.
- Dark Theme (Focus): Background color: Charcoal (#1C1C1C) for an elevated card and container surface.
- Dark Theme (Focus): Accent color: Mint (#8FB59E) for achievements.
- Headings: 'Bebas Neue' style: Bold, condensed, uppercase sans-serif for headings.
- Body text: 'Inter': Clean, high-legibility sans-serif for body text.
- Consistent icon set for bottom tab bar: 'Arena', 'Compete', 'Quests', 'Feed', and 'More'.
- Rounded-corner cards (16px radius) with thin borders (1px) for inactive states.
- Pill-style buttons and segmented progress bars.
- Subtle animations for transitions between screens and feedback for correct/incorrect answers.
